# 🚗 AutoHireX – Car Rental & Admin Dashboard System

AutoHireX is a full-stack car rental platform designed to provide a seamless vehicle booking experience with a powerful admin dashboard for managing inventory and reservations.

---

## 🌟 Features

### 👤 User Features
- 🚘 Browse available cars with detailed information  
- 📍 Select pickup location and booking dates  
- 📅 Real-time availability checking  
- 🔐 Secure authentication (Signup/Login using JWT)  
- 📊 User dashboard to manage bookings  

---

### 🛠️ Admin Features
- 📦 Add, update, and delete car listings  
- 📈 Manage bookings and users  
- 🧾 Monitor reservation activity  
- ⚙️ Fully functional admin dashboard  

---

## 🔐 Authentication & Security
- JWT-based authentication system  
- Protected routes for users and admin  
- Secure API communication  

---

## 🖼️ Image Optimization
- Integrated **ImageKit** for:
  - Fast image loading  
  - Real-time transformations  
  - Optimized media delivery  

---

## ⚡ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT  
- **Media:** ImageKit  
- **API:** REST API  

---

## 📦 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/shivamkrashish/AutoHireX.git

# Navigate to project folder
cd AutoHireX

# Install frontend dependencies
cd client
npm install

# Install backend dependencies
cd ../server
npm install

# Run backend server
npm run server

# Run frontend
cd ../client
npm start
